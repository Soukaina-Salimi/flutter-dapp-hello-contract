Ce projet est une DApp Flutter qui interagit avec un smart contract Solidity déployé sur Ganache.
Il montre comment :

    Charger l’ABI du contrat et son adresse
    
    Appeler une fonction view (lecture) comme yourName()
    
    Envoyer une transaction setName() pour modifier l’état du contrat
    
    Afficher les données dans une interface Flutter
    
    Utiliser web3dart pour la communication Web3

Le smart contract HelloWorld.sol contient une simple gestion de chaîne de caractères, idéale pour comprendre les bases de l’interaction Flutter ↔ Ethereum.

Le projet peut servir de template pour construire des applications décentralisées Flutter plus avancées (gestion de tokens, NFT, stockage décentralisé, etc.).
